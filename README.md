# Task Tracer
A task tracking application built as a personal project to help develop my Node.js skills.
## Technologies used
### Client
- React
- TypeScript
- Vite
- ESLint
- Mantine (UI)
### Server
- Node.js
- Express
### Database
- PostgreSQL
- Prisma
### Authentication
- JSON Web Tokens (JWT) for secure authentication
- bcrypt for password hashing and encryption
### Deployment
- Render
